How To Borrow & Return

ขั้นตอนการยืมวิทยุ

1

เลือกเครื่องที่ว่าง

แตะที่การ์ดวิทยุที่มีสถานะ 🟢 ว่าง เพื่อเปิดแบบฟอร์มยืม

2

กรอกรหัสพนักงานและชื่อ

กรอกรหัสพนักงาน และชื่อ-นามสกุลของคุณให้ครบถ้วน

3

กด "ลงทะเบียนขอยืม"

ระบบจะบันทึกข้อมูลทันที วิทยุจะเปลี่ยนสถานะเป็น 🔴 ถูกยืม

ขั้นตอนการคืนวิทยุ

1

ไปที่แท็บ "สถานะของฉัน"

ดูรายการวิทยุที่คุณกำลังถือครองอยู่

2

กด "แจ้งคืน"

วิทยุจะเปลี่ยนเป็นสถานะ 🟡 รอตรวจรับ และแจ้ง Admin โดยอัตโนมัติ

3

Admin ยืนยันรับคืน

เมื่อ Admin กดยืนยัน วิทยุจะกลับสู่สถานะ 🟢 ว่าง พร้อมให้ยืมใหม่

ข้อมูลการยืม-คืนจะถูกบันทึกและซิงค์แบบ Real-time ทุกอุปกรณ์ ไม่ต้องรีเฟรชหน้า
